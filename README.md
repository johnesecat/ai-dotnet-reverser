# AI .NET Deobfuscator

🔬 Advanced browser-based .NET assembly deobfuscation tool supporting **17 different obfuscators**.

## Supported Obfuscators

1. ✅ **ConfuserEx** - String encryption, control flow, renaming
2. ✅ **Dotfuscator** - Sequential renaming, enhanced methods
3. ✅ **SmartAssembly** - String encoding, resource embedding
4. ✅ **.NET Reactor** - NecroBit, anti-tampering
5. ✅ **Agile.NET** - Virtualization, CLI protection
6. ✅ **Crypto Obfuscator** - Symbol renaming, ILDASM protection
7. ✅ **Eazfuscator.NET** - Virtualization, resource encryption
8. ✅ **Babel Obfuscator** - Control flow, string encryption
9. ✅ **ILProtector** - Native protection, anti-debug
10. ✅ **Phoenix Protector** - Heavy virtualization
11. ✅ **MaxtoCode** - Encryption, anti-debug
12. ✅ **DNGuard** - HVM virtualization
13. ✅ **Goliath** - Renaming, string encryption
14. ✅ **DeepSea** - Heavy renaming, control flow
15. ✅ **Obfuscar** - Simple sequential renaming
16. ✅ **Xenocode (Spices.Net)** - String encryption, renaming
17. ✅ **Unknown Obfuscator** - Heuristic detection

## Features

- 🌐 **100% Browser-Based** - No backend required
- 🚀 **Fast Analysis** - Instant obfuscator detection
- 📊 **Detailed Statistics** - Comprehensive metrics
- 🎨 **Modern UI** - Dark mode, responsive design
- 🔒 **Privacy-First** - All processing in your browser
